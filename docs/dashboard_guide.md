# Dashboard Guide

The Streamlit console is the primary local review surface.

Pages:
- `Overview`: latest regime, rankings, and basket snapshot
- `What Changed`: state transitions and proxy deterioration notes
- `Baskets`: candidate baskets and simple research summaries
- `Exposure`: regime-based exposure overlay and score history
- `Proxy Health`: validation and unstable proxy review
- `Run History`: persisted run timeline
- `Alerts`: latest alerts, alert history, and watchlist
- `Exports`: quick file-level export inventory

Launch with:
`bash scripts/start_dashboard.sh`
