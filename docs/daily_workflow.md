# Daily Workflow

1. Run `bash scripts/run_daily.sh`.
2. Review `outputs/reports/daily_report.md`.
3. Open the dashboard with `bash scripts/start_dashboard.sh`.
4. Check the `What Changed`, `Baskets`, and `Alerts` pages first.
5. Export snapshots from `outputs/exports/` for manual chart or note workflows.

Daily runs persist:
- run metadata
- agent scores
- rankings
- alerts
- baskets
- transitions
